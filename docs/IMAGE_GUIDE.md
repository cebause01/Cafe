## Image Specifications:
- **Recommended size**: 800x800px or 1000x1000px (square format works best)
- **Format**: PNG or JPG
- **Aspect ratio**: 1:1 (square) recommended
- **File size**: Keep under 500KB for web performance

## Where to Find Images:
- Unsplash (unsplash.com) - Free high-quality photos
- Pexels (pexels.com) - Free stock photos
- Pixabay (pixabay.com) - Free images
- Generate using AI tools (like the one used for Humble Mornings Blend)

## How to Add Images:
1. Save images to the `picture/` folder
2. Update the `image` property in `shop.js` (around line 20, 30, 38, etc.)
3. Update the `image` property in `cart.js` (around line 104, 113, 122, etc.)
4. The images will automatically appear on the shop page, cart, and modals!
